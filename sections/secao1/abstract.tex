\begin{abstract}
    O algoritmo ``Instance Selection with Ant Colony Optimization'' se destacou ao comparar com outros algoritmos de seleção de instâncias 
    como o {K-means} ou {Support Vector Machine}. Uma versão paralela mostra-se essencial para grandes conjuntos de dados que 
    levariam muito tempo para serem processados de forma sequencial. Uma versão paralela em GPU usando CUDA está sendo contruida
    e ganhos de desempenho significativos são esperados na versão final do algoritmo.
\end{abstract}
